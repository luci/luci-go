# Fleet Console Agent Skills

This directory contains skills (instructions and guidelines) for AI agents working on the Fleet Console frontend.

## Available Skills

- [prepare_cl](./skills/prepare_cl/SKILL.md): Instructions for preparing a CL for upload, including committing, verification, and optional upload.
- [senior_reviewer](./skills/senior_reviewer/SKILL.md): Instructions for running the SeniorReviewer subagent for self-review.
- [manual_testing](./skills/manual_testing/SKILL.md): Guidelines for manual testing.
- [aip160_filtering](./skills/aip160_filtering/SKILL.md): Guidelines for AIP-160 filtering.
- [high_density_ui](./skills/high_density_ui/SKILL.md): Guidelines for high density UI design.
- [project_verification](./skills/project_verification/SKILL.md): Guidelines for project verification (lint, test, etc.).
- [ux_prototyping](./skills/ux_prototyping/SKILL.md): Guidelines for UX prototyping.

## Usage

Agents should read the `SKILL.md` file for the relevant skill before performing tasks associated with that skill.
