# Generic Tools

Utility functions and classes.

## Categories

*   [**pRPC**](./prpc.md): Clients and utilities for making pRPC requests.
*   [**Async Patterns**](./async_patterns.md): Utilities for handling
    asynchronous operations (e.g., `batchedFn`, `cachedFn`).
*   [**Observers**](./observers.md): Tools for observing DOM elements (e.g.,
    `ObserverElement`).
*   [**Utils**](./utils.md): String, number, and iteration utilities.
