# Generic Hooks

Reusable React hooks for common functionality.

## Categories

*   [**Data Fetching**](./data_fetching.md): Hooks for efficient data fetching
    and pagination (e.g., `useVirtualizedQuery`).
*   [**State Management**](./state_management.md): Hooks for managing state,
    including URL search params `useSyncedSearchParams`.
*   [**Utils**](./utils.md): General utility hooks (e.g., `useSingleton`,
    `useIsDevBuild`).
