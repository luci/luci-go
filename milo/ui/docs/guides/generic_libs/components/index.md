# Generic UI Components

Use these components to build consistent and feature-rich UIs.

## Categories

*   [**Routing & Navigation**](./routing_and_navigation.md): Components for
    handling routing, tabs, and navigation (e.g., `RoutedTabs`).
*   [**Editors & Inputs**](./editors_and_inputs.md): Rich text editors and
    input components (e.g., `CodeMirrorEditor`, `TextAutocomplete`).
*   [**Layout & Display**](./layout_and_display.md): Layout utilities and
    display components (e.g., `QueuedSticky`, `ExpandableEntry`, `Table`).
*   [**Interaction**](./interaction.md): Components for user interaction (e.g.,
    `Hotkey`, `DragTracker`, `CopyToClipboard`).
*   [**Analytics**](./analytics.md): Google Analytics integration components.
*   [**Lit Integration**](./lit_integration.md): Utilities for integrating Lit
    components with React and MobX.
